Project

XPlore UCSD



Description 
Your personal virtual tour guide for visiting the UC san Diego Campus.

XPlore UCSD is a mobile application that aims to provide a pleasant, 

easy-to-use, and highly customizable guided touring experience for all tourists 

and prospective students visiting the UC San Diego campus.


XPlore UCSD on App Store | XPlore UCSD on Google Play

Using SwiftUI and Android Studio, I implemented:

  1. Tour page: Users can plan the stops for any tours; users can go front and back to any stops they want within the page, and the page updates itself with corresponding info; link backend JSON files of preset tours to UI; built-in audio player and texts for introduction to the stops.
  2. Search page: users can search using my customized algorithm based on name or amenities filter to get the landmarks they want.
  3. Splash and Onboarding page: intro of app to first time users.
  4. Directions function using Google / Apple Maps API
